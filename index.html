IST 652 Scripting for Data Analysis
2025

NBA Seasonal Trend Analysis and Exploration

I worked in a group to look at NBA seasonal trends. For this project, we loaded data from the NBA website, which culminated in around 14,000 unique player seasons. We sought to find trends in NBA three-point attempts by play style over time, statistical peaks by player age, and three-point attempts over time. To accomplish these tasks, we used descriptive statistics, visualizations, and unsupervised learning techniques. For the unsupervised learning, we used k-means clustering to cluster the data into three groups, which were then labeled as star players, big men, and role players. Then we plotted the change in three-pointers over a 20-year period. The results were quite interesting, with star players seeing the highest increase in three-point attempts over the period. Role players had a more gradual increase, with big men seeing a very minimal increase. One thing to note about this grouping was that star big men were clustered as star players, with the star big men actually seeing an increase in attempts. Another piece of the analysis was the statistical peaks by age. In this section we looked at which ages were better for certain statistics. One of the takeaways from this section was that playmaking is a skill where players will peak later on in their careers. This makes sense since playmaking requires a certain amount of basketball intelligence, which can be learned over time. Scoring, on the other hand, is a skill set that tends to be more suited for younger players who may be more athletic. We also saw that over the 20-year period, three-point attempts increased significantly. Meanwhile, the average amount of field goal attempts did not increase by very much.

Assessing the Predictability of Maximum Heart Rate Using Demographic and Cardiovascular Metrics

For this project, I was challenged to use a research paper-based approach that incorporated both Frequentist and Bayesian statistical methods to uncover insights in data. The research question I chose was to determine if maximum heart rate could be accurately predicted using other available health data. The data I chose had around 1,000 rows with features that included demographic metrics like age and gender, as well as cardiovascular metrics like resting blood pressure and levels of serum cholesterol. I then used a Frequentist linear regression model that had maximum heart rate as the dependent variable and age, gender, exercise-induced angina, ST depression, slope of ST during exercise, number of major vessels colored by fluoroscopy, and thalassemia test result. Together, these variables resulted in a model R² of 0.345, meaning that 34.5% of the variance in heart rate can be explained by the model. The Bayesian model kept the same variable and had a Bayes factor of 3.8 x 10⁸⁵, which provides strong evidence that the independent variables improve the prediction of maximum heart rate compared to a model with no predictors. Both of these methods show that maximum heart rate is correlated to health and demographic variables, and it is possible to build an accurate model if given the right predictors. This would make getting maximum heart rate values much easier for the patient since they wouldn’t have to do intense exercise to get their heart rate up. This project allowed me to apply my knowledge of different modeling methods to real health problems with the guidance of my professor.

IST 691 Deep Learning in Practice
2025

Emotion-Driven Music Recommendation Using Real-Time Facial Expression Recognition

For this group project, we created a system that recommended songs based on the users' emotions using facial data. To complete this project, we got lots of labeled facial data based on emotions like happy, sad, and disgust. Then we trained our own convolutional neural network to classify new images into emotion categories. Then we assigned each emotion a valence score, which measures emotional stimulus. This was then applied to a dataset that had numerous songs with valence scores. The system would then match the valence score of the classified emotion with the accompanying song. You could use this system either live with a webcam or by uploading a photo. This project taught me a lot about how to build my own convolutional neural network using TensorFlow and Keras in Python. We then presented this project at the iSchool poster day.

Learning Outcome: 2, 3, 4, 6
